# Company Handbook

## Approval Thresholds

| Category        | Threshold  | Approval Level |
| --------------- | ---------- | -------------- |
| Email (send)    | Any        | Standard       |
| Financial       | Any amount | Elevated       |
| Social Media    | Any post   | Standard       |
| Account Changes | Any        | Elevated       |
| Data Export     | Any        | Elevated       |

## Working Hours

- Default: 24/7 autonomous for non-sensitive tasks
- Sensitive actions: Human approval required regardless of time

## Escalation Contacts

- Primary: Human operator (vault owner)
